## Terminal Usage

Prof's directions -> for windows

For Mac

1. Cmd + D -> split screen

2. Cmd + Shift + D -> exit split screen

3. Ls -vCFG -> prints ls contents in color to differentiate files with folders

4. Head -23 FILENAME.file -> opens file in terminal

5. emacs FILENAME.org -> ctrl c + ctrl e + h + o -> html file made

6. for emacs when in doubt-> cmd x + cmd c (exits emacs entirely, but confirms save beforehand)

## Myhelp Usage
